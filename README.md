# Dark Mode To-Do App
A modern dark-themed To-Do List built using HTML, CSS, and JavaScript.

# Features
- Add tasks
- Mark tasks as completed
- Line-through effect on completed tasks
- Remove completed tasks
- Smooth animations
- Interactive status messages
- Dark modern UI

# Technologies Used
- HTML
- CSS
- JavaScript
